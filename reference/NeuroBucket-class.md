# NeuroBucket

a four-dimensional image that consists of a sequence of labeled image
volumes backed by a list

## Slots

- `labels`:

  the names of the sub-volumes contained in the bucket

- `data`:

  a list of
  [`NeuroVol`](https://bbuchsbaum.github.io/neuroim2/reference/NeuroVol-class.md)
  instances with names corresponding to volume labels
