# NeuroVol Class

Base class for representing 3D volumetric neuroimaging data. This class
extends
[`NeuroObj`](https://bbuchsbaum.github.io/neuroim2/reference/NeuroObj-class.md)
to provide a foundation for various types of 3D brain images.

## Details

NeuroVol serves as an abstract base class for more specific 3D
neuroimaging data structures. It inherits spatial properties from
NeuroObj but does not specify a particular data storage method.
