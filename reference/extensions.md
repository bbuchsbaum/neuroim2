# Get Extensions from an Object

Generic function to retrieve NIfTI extensions from various object types.

## Usage

``` r
extensions(x, ...)
```

## Arguments

- x:

  An object potentially containing extensions.

- ...:

  Additional arguments (currently unused).

## Value

A
[`NiftiExtensionList-class`](https://bbuchsbaum.github.io/neuroim2/reference/NiftiExtensionList-class.md)
object, or NULL if no extensions.
