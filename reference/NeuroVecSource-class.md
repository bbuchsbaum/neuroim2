# NeuroVecSource Class

A class used to produce a
[`NeuroVec`](https://bbuchsbaum.github.io/neuroim2/reference/NeuroVec-class.md)
instance.

## Slots

- `indices`:

  An `integer` vector representing the indices of the volumes to be
  loaded.
