# Check if Extensions are Present

Tests whether an object has any NIfTI extensions.

## Usage

``` r
has_extensions(x)

# S4 method for class 'NiftiExtensionList'
has_extensions(x)

# S4 method for class 'list'
has_extensions(x)
```

## Arguments

- x:

  An object to test.

## Value

Logical indicating whether extensions are present.
